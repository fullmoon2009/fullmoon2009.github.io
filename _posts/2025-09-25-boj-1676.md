## 문제
N!에서 뒤에서부터 처음 0이 아닌 숫자가 나올 때까지 0의 개수를 구하는 프로그램을 작성하시오.
<br><br>


## 입력
첫째 줄에 N이 주어진다. (0 ≤ N ≤ 500)
<br><br>


## 출력
첫째 줄에 구한 0의 개수를 출력한다.
<br><br>


## 아이디어
곱 전체에 들어있는 5의 개수를 구하면 되는 문제
<br><br>

## 풀이
```python
import sys

N = int(sys.stdin.readline())
ans = 0
d = 5
while d <= N:
    ans += N // d
    d *= 5
print(ans)
```
